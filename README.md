# Ledger Wallet -- QA Automation Engineer Onboarding Guide

> **Audience**: New QA Automation Engineer interns joining the Ledger Live team.
>
> **Structure**: Five progressive parts, from foundations to mastery, plus crash courses and appendices. Each chapter builds on the previous one. Do not skip ahead.
>
> **Quizzes**: Each chapter ends with an interactive quiz. Each part ends with a final assessment. You need **80%** to pass.
